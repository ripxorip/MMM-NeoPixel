# Module: NeoPixel
(WIP)
The `NeoPixel` Is used to control a NeoPixel led strip attached to an Arduino
## Using the module
(WIP)
To use this module, add it to the modules array in the `config/config.js` file:
````javascript
modules: [
	{
		module: "MMM-NeoPixel",
	}
]
````
